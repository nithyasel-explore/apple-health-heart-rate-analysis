# Apple Health Heart Rate Analysis

This repository contains an exported HTML notebook analyzing Apple Watch heart rate data.

## Files
- `index.html` — annotated notebook export
- `manifest.json` — basic web app manifest for browser install support
- `.gitignore` — standard Python/Jupyter ignores for future notebook work

## Notes
This project focuses on:
- isolating Apple Watch heart rate records
- cleaning datetime and numeric fields
- estimating daily resting heart rate from lower heart-rate observations
- visualizing long-term trends and medication timing
